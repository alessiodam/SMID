For Developers

This tool helps you retrieve your SmartSchool user ID using your session cookie.

How to use:

  1. Log in to your SmartSchool account
  2. Get your PHPSESSID cookie value (using browser developer tools)
  3. Enter your school's subdomain and the PHPSESSID in the form
  4. Submit to retrieve your user ID

API Endpoints:

Request authentication code:

GET https://smid.alessiodam.dev/v1/auth-code?domain=YOUR_SMARTSCHOOL_DOMAIN&phpSessId=YOUR_PHPSESSID

Request user ID:

GET https://smid.alessiodam.dev/v1/user-id?code=THE_AUTH_CODE

Security Note:

Your subdomain & PHPSESSID is sensitive information. This tool stores a hashed version of your PHPSESSID for caching purposes, but you should still be careful when sharing it.

SMID is an unofficial tool and is not affiliated with SmartSchool.
